Este projeto é uma aplicação web construída em Java utilizando Spring Boot. O sistema permite cadastro e listagem de produtos, clientes e pedidos, controle de login e logout de usuários, bem como gerenciamento de sessão para diferenciar perfis comuns e administradores.
A camada web foi implementada com Spring MVC e páginas JSP, usando JSTL para renderização dos dados. A persistência é feita com Spring Data JPA e Hibernate, tendo como o banco de dados o PostgreSQL, com pool de conexões configurado via HikariCP.
O projeto também conta com carregamento inicial de dados a partir de arquivos texto utilizando ApplicationRunner, facilitando testes e simulações. Para testes automatizados, é utilizado JUnit 5 com Spring Boot Test, garantindo que o contexto da aplicação seja carregado corretamente.
